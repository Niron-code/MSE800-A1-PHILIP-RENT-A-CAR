A secure, user-friendly car rental management system for customers and administrators

Philip Rent-A-Car is a modern car rental management system designed to make vehicle booking and administration simple, secure, and efficient. Whether you are a customer looking to browse available cars, make a booking, or an administrator managing rentals and fleet details, this system provides a seamless experience with role-based access, secure authentication, and an intuitive interface.

Key Features

  • User-friendly signup and login for both customers and admins
  • Search and filter cars by model, year, price, and availability
  • Transparent rental booking process with real-time cost calculation
  • Admin dashboard for managing cars, approving or rejecting rental requests, and updating availability
  • Secure handling of rental records and customer information

This documentation provides detailed guidance on system functionality, installation, database structure, and usage examples.
